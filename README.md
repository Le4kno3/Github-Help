# Github-Help

1. Shortcut to git add and commit: 
 * `git config --global alias.add-commit '!git add -A && git commit'`
 * Now to use this run on the project, `git add-commit -m 'My commit message'`

2. Change branch
 * `git branch -m BRANCHNAME`
